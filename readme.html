üìù Overview

This document describes a Python-based program that resizes grayscale images using the nearest neighbor interpolation technique. The user provides the image path and a scaling factor. The script displays both the original and resized images.

üì¶ Requirements

The program depends on the following Python libraries:

  • Pillow (PIL): For loading and displaying images
  • NumPy: For array-based image processing
pip install pillow numpy

üîç Description of Functionality

  1. Image Loading and Conversion: The image is converted to grayscale for simplified processing.
  2. Dimension Extraction: The original image dimensions are obtained for resizing calculations.
  3. Resizing Logic: Nearest neighbor logic maps each new pixel to its closest counterpart in the original image.
  4. Pixel Mapping: A new image is created and filled with the selected pixel values.
  5. Display: Both original and resized images are displayed for visual confirmation.

üß† How Nearest Neighbor Works

For each pixel in the resized image, the algorithm identifies and copies the value of the closest pixel in the original image. This method is fast but can produce visible blocky artifacts, especially when enlarging images.
